# 实习经历目录

本目录用于存放您的实习经历条目。

## 添加实习经历

1. 在本目录下创建新的Markdown文件
2. 文件命名建议格式：`年份-公司名称.md`（例如：`2024-abc公司.md`）
   也可使用原格式：`YYYY-MM-DD-internship-name.md`

3. 使用以下YAML前置元数据格式：
   ```yaml
   ---
   layout: archive
   title: "[公司名称] 实习经历"
   date: YYYY-MM-DD
   description: "简短描述实习内容"
   location: "工作地点"
   company: "公司名称"
   position: "实习职位"
   link: "" # 可选，公司官网链接
   ---
   ```

4. 在前置元数据后添加详细的实习经历内容

## 示例结构

```markdown
## 实习概述

在[公司名称]担任[实习职位]期间，参与了[项目名称]项目的开发和实施工作。主要负责[详细工作内容描述]。

## 主要职责

* 负责[具体职责1]
* 参与[具体职责2]
* 协助[具体职责3]

## 工作成果

* 完成了[具体成果1]
* 开发了[具体成果2]

## 收获与成长

通过这次实习，我深入了解了[行业/技术领域]，掌握了[具体技能]。
```

## 注意事项

- 所有实习条目将在网站的"实习实践"页面中按年份分组显示
- 确保文件编码为UTF-8以正确显示中文
- 可以添加Markdown格式的图片、链接等丰富内容
- 请参考目录中的示例文件作为模板
